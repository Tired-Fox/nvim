# Neovim Config

- Install the following
    - Neovim v0.10 with `bob` which can be installed with `packam -S bob`
    - Have `node`, `go`, `rust`/`cargo`, `zls` installed
        - `zls` with `zvm`
        - `node` with `fnm`
    - [fzf](https://github.com/junegunn/fzf)
    - ripgrep: with `cargo install`
